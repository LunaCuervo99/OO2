# Ejercicio 6: 

Para cada una de las siguientes situaciones, realice en forma iterativa los siguientes pasos:
(i) indique el mal olor,
(ii) indique el refactoring que lo corrige, 
(iii) aplique el refactoring, mostrando el resultado final (código y/o diseño según corresponda). 
Si vuelve a encontrar un mal olor, retorne al paso (i). 

## 6.2 Juego
```java
 public class Juego {
    // ......
    public void incrementar(Jugador j) {

        j.puntuacion = j.puntuacion + 100;

    }

    public void decrementar(Jugador j) {

        j.puntuacion = j.puntuacion - 50;

    }


public class Jugador {
    public String nombre;
    public String apellido;
    public int puntuacion = 0;
}
```
### Solución:
* **(i) Mal olor:** **Public Field **: La clase `Jugador` tiene sus atributos públicos, lo que rompe el encapsulamiento.
* **(ii) Refactoring:** **Encapsulate Field**: Ponemos los atributos privados y hacemos metodos getters/setters si corresponde.
* **(iii) Resultado:**
```java
public class Juego {
    // ......
    public void incrementar(Jugador j) {

        j.puntuacion = j.puntuacion + 100;

    }

    public void decrementar(Jugador j) {

        j.puntuacion = j.puntuacion - 50;

    }

public class Jugador {
    private String nombre;
    private String apellido;
    private int puntuacion = 0;

    public int getPuntuacion() {
        return this.puntuacion;
    }

    public void setPuntuacion(int puntuacion) {
        this.puntuacion = puntuacion;
    }
}
```

* **(i) Mal olor:** **Feature Envy**: La clase `Juego` esta usando atributos de Jugador, en vez de sus propios atributos. Los métodos incrementar y decrementar corresponden a `Jugador`
* **(ii) Refactoring:** **Move Method**: Pasamos los métodos a la clase que corresponde.
* **(iii) Resultado:**
```java
public class Juego {
    // ......
    public void incrementar(Jugador j) {
        j.sumarPuntos(100);
    }

    public void decrementar(Jugador j) {
       j.restarPuntos(50);
    }

public class Jugador {
    private String nombre;
    private String apellido;
    private int puntuacion = 0;

    public int getPuntuacion() {
        return this.puntuacion;
    }

    public void setPuntuacion(int puntuacion) {
        this.puntuacion = puntuacion;
    }

    public void sumarPuntos(int puntos) {
        this.puntuacion += puntos;
    }

    public void restarPuntos(int puntos) {
        this.puntuacion -= puntos;
    }
}
```
